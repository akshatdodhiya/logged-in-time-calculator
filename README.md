# Logged-in Time Calculator
A simple yet useful java program that calculates the amount of time a user is logged in to a system. It differentiates different users by a unique ID which user himself has to enter while running the program(when asked).

## Features
* Calcuates number of days, hours and minutes between user's login and logout.
* Displays output in a clean manner.

## Usage
The file can run in any machine which has Java installed in it.
